📄 节税攻略核心要点：
1. “扣税三宝”组合拳：
	- QDAP（延期年金）与 TVC（强积金自愿供款）：每年共享 $60,000 扣税额。对于高薪人士，这能直接省下约 $10,200 的税款。

3. 家庭免税额的“最优配置”：
	- 供养父母/祖父母：如果父母年满 60 岁且与你同住，免税额翻倍（$100,000）；如果不通住但每年给予不少于 $12,000 的生活费，仍可享有 $50,000 免税额。

4. 合并计税 (Joint Assessment) 的判定：
	- 并不是所有夫妻合并计税都省钱。通常只有在**一方高薪、一方低薪（甚至无薪）**的情况下，合并计税利用未用尽的免税额才具节税效应。

## 一、核心扣税工具深度解析

本章节将逐一详解香港薪俸税体系下四种最主流的扣税工具：自愿医保（VHIS）、合资格延期年金（QDAP）与强积金自愿性供款（TVC）、居所贷款利息（Home Loan Interest）以及住宅租金扣除（Domestic Rent Deduction）。我们将以表格和要点形式，清晰呈现各项工具的扣税逻辑、最新额度、适用条件与合规要点。

### 1. 自愿医保计划（VHIS）：为自己与家人的健康投资减税

自愿医保计划是一项由政府认证的个人住院保险产品。为自己或“指明亲属”购买合资格的 VHIS 保单，所支付的保费可用于扣除薪俸税或个人入息课税。

**核心优势**：纳税人可为多名指明亲属（如配偶、子女、父母、祖父母、兄弟姐妹）投保，每名受保人的保费均可享受独立的扣税额，实现家庭成员健康保障与税务优化的双重目标。

#### **关键要素速览**

|   |   |
|---|---|
|**要素**|**规则与说明**|
|**扣除上限**|• **每名受保人每年最高 HK$8,000**（截至 2024/25 评税年度）。 • 此上限独立计算，不与其他扣除项目共享。|
|**适用人群（受保人）**|• 纳税人本人。 • 纳税人的“指明亲属”，包括：<br><br>- 配偶与子女。<br>    <br>- 纳税人或其配偶的**父母、祖父母、外祖父母、兄弟姐妹**。|
|**申报主体（保单持有人）**|• 必须是纳税人本人或其配偶（非分居）。 • 即使保费由其他家庭成员（如子女为父母）支付，也必须由符合资格的保单持有人提出申索。|
|**合规要点**|1. **认证计划**：必须是经香港特区政府医务卫生局认证的“自愿医保计划”产品。<br>    <br>2. **保费收据**：须保留由保险公司出具的保费收据及年度结单，作为申报依据。<br>    <br>3. **不可重复申报**：已获雇主或其他人士/机构报销的保费部分，不可再申请扣税。<br>    <br>4. **按实额扣除**：扣除额为**实际支付的净保费**，上限为 HK$8,000。若实际保费低于此数，则以较低者为准。|

#### **家庭成员叠加与申报逻辑**

VHIS 的一大特点是其“家庭叠加”效应。一位纳税人可以为多名指明亲属购买保单，并为每位受保人分别申索最高 HK$8,000 的扣除额。

**示例：**

王先生（纳税人）为自己、妻子、一名子女及自己的父亲购买了 VHIS 认证计划，保费情况如下：

- **王先生本人**：年保费 HK$10,000
    
- **王太太**：年保费 HK$6,000
    
- **儿子**：年保费 HK$2,000
    
- **王先生的父亲**：年保费 HK$12,000
    

在 2024/25 评税年度，王先生可申索的 VHIS 扣除总额为：

- 为自己申索：**HK$$8,000**（虽支付了 H$$10,000，但以 HK$8,000 为上限）
    
- 为妻子申索：**HK$6,000**（按实际支付额）
    
- 为儿子申索：**HK$2,000**（按实际支付额）
    
- 为父亲申索：**HK$$8,000**（虽支付了 H$$12,000，但以 HK$8,000 为上限）
    
- **合计可扣除金额** = 8000 + 6000 + 2000 + 8000 = **HK$24,000**
    

#### **常见问答（FAQ）**

**Q1：如果我和哥哥共同为父亲购买一份 VHIS 保单，如何申索扣税？**

**A1**：如果一份保单有多名保单持有人，保费将被视为由所有持有人平均分摊。例如，保单年费 HK$$12,000，您和哥哥均为持有人，则每人可申索 H$$6,000 的扣除。

如果你们各自为父亲购买了不同的 VHIS 保单，则可以分别就自己支付的保费进行申索，但每人就同一位受保人（父亲）的申索上限仍为 HK$8,000。

**Q2：为家人购买 VHIS，与我是否申索了供养父母/子女免税额有冲突吗？**

**A2**：没有冲突。VHIS 保费扣除与供养父母/祖父母免税额或子女免税额是两项独立的税务宽减，可以同时申领。只要符合各自的条件即可。

**Q3：我为配偶支付了 VHIS 保费，但我们选择分开报税，我可以申索这笔扣除吗？**

**A3**：可以。即使夫妻双方选择分开评税，只要你们没有分居，一方为另一方支付的 VHIS 保费，支付方依然可以申索扣除。夫妻二人也可以协议由其中一方申索全部扣除额，或按任意比例分配，但总额不能超过实际支付的保费，且每名受保人的上限为 HK$8,000。

**Q4：受保的家人必须是香港居民吗？**

**A4**：是的。根据税务局规定，享受 VHIS 保费扣除的受保人必须是香港居民（即持有香港身份证）。

### 2. 合资格延期年金（QDAP）与强积金自愿性供款（TVC）：规划退休，享受即时税务优惠

为了鼓励市民为退休生活进行储蓄，政府推出了两项具备税务扣除功能的工具：合资格延期年金保费（QDAP）和强积金可扣税自愿性供款（TVC）。这两项开支合并计算扣除上限。

**核心优势**：纳税人可以通过这两项工具，在为未来退休做准备的同时，直接降低当年的应课税入息，尤其适合希望强制储蓄并享受稳定税务优惠的职场人士。

#### **关键要素速览**

|   |   |
|---|---|
|**要素**|**规则与说明**|
|**合并扣除上限**|• **每名纳税人每年最高 HK$60,000**（截至 2024/25 评税年度）。 • 此上限为 **QDAP 与 TVC 两项的总和**。|
|**适用人群**|• **QDAP**：合资格延期年金保单的年金领取人必须是纳税人本人或其配偶，且在相关课税年度内持有香港身份证。 • **TVC**：必须是强积金计划下的“可扣税自愿性供款账户”的持有人。|
|**申报主体**|• **QDAP**：保单持有人（纳税人或其配偶）。 • **TVC**：供款的账户持有人。|
|**合规要点**|1. **认证产品/账户**：QDAP 须为保险业监管局认证的产品；TVC 须为强积金计划下明确设立的“可扣税自愿性供款账户”。<br>    <br>2. **分清供款类型**：普通的强积金强制性供款、特别自愿性供款（SVC）均**不可扣税**。只有投入 TVC 账户的部分才符合资格。<br>    <br>3. **证明文件**：须保留由保险公司出具的“合资格延期年金保费的年度摘要”或由强积金受托人出具的“可扣税自愿性供款概要”，作为申报依据。<br>    <br>4. **夫妻合并申报**：已婚人士可灵活分配 QDAP 的扣除额，但每人申索的上限仍为 HK$$60,000，夫妻合计申索总额不得超过两人支付的保费总和及 H$$120,000。TVC 供款则不可分配。|

#### **QDAP vs. TVC：差异与选择**

虽然共享扣除上限，但 QDAP 和 TVC 在性质、流动性及投资方式上存在显著差异。

|   |   |   |
|---|---|---|
|**对比维度**|**合资格延期年金（QDAP）**|**强积金可扣税自愿性供款（TVC）**|
|**产品性质**|一种长期保险产品，由保险公司提供。|一种在强积金制度下的自愿性供款。|
|**资金锁定**|至少锁定至 60 岁或指定退休年龄才可开始领取年金。|锁定至 65 岁（或法定的提早退休年龄）方可提取。|
|**投资方式**|通常由保险公司管理投资，提供保证及/或非保证回报。|供款人可自行选择强积金计划下的核准基金进行投资。|
|**费用结构**|通常包含保费费用、管理费等，内置于产品中。|涉及基金管理费、行政费等，与标准强积金基金类似。|
|**适用人群**|适合偏好稳定、不善投资，希望有保证回报的储蓄者。|适合熟悉基金投资，希望自主管理退休资产的供款人。|

#### **夫妻如何分配更划算？**

QDAP 的一个税务优势在于夫妻间的灵活性。如果夫妻一方收入较高，另一方收入较低或无收入，高收入方可以利用未用尽的扣除额度。

**规则**：如果夫妻双方选择**合并评税**或**个人入息课税**，他们可以申请重新分配两人支付的合资格年金保费的扣除。

**示例：**

陈先生年收入 HK$800,000，陈太太为家庭主妇，无应课税收入。

- **情景一：仅陈先生供款**
    
    - 陈先生为自己购买 QDAP，年缴保费 **HK$70,000**。
        
    - 他个人最多只能申索 **HK$60,000** 的扣除额。
        
- **情景二：夫妻共同规划**
    
    - 陈先生为自己购买 QDAP，年缴保费 **HK$60,000**。
        
    - 陈先生同时作为保单持有人，为陈太太购买一份 QDAP，年缴保费 **HK$60,000**。
        
    - 在选择**个人入息课税**或**合并评税**的情况下，陈先生可以申索自己支付的 HK$$60,000，并同时申索为陈太太支付的 H$$60,000。
        
    - **陈先生合计可扣除金额** = 60000 + 60000 = **HK$120,000**。
        

**重点提示**：TVC 供款不具备此灵活性，**只能由账户持有人本人申索扣除**，不可转移给配偶。因此，对于希望最大化利用夫妻二人共 HK$120,000 扣除总额的家庭，优先考虑配置 QDAP 会更具弹性。

#### **常见问答（FAQ）**

**Q1：如果我同时购买了 QDAP 和 TVC，扣除顺序是怎样的？**

**A1**：税务局会**优先扣除 TVC**，然后再扣除 QDAP 的保费。例如，您支付了 HK$$40,000 TVC 和 H$$50,000 QDAP，合计 HK$$90,000。在 H$$60,000 的上限下，会先扣除全部 HK$$40,000 的 TVC，再扣除 H$$20,000 的 QDAP。

**Q2：对于收入不高、税率较低的人来说，做 TVC 或 QDAP 是否划算？**

**A2**：这取决于个人财务目标。如果您处于较低的边际税率（如 2% 或 6%），通过这两项工具获得的即时节税金额有限。其主要价值可能更多在于强制储蓄以应对退休。如果您处于较高的边际税率（如 17%），每投入 HK$$60,000 最多可节省 H$$10,200 的税款，税务诱因则非常显著。

**Q3：我可以在年中随时开始做 TVC 吗？**

**A3**：可以。您可以随时联系您的强积金受托人开立 TVC 账户，并根据自己的现金流安排，选择一笔过或分期供款。为确保能在当个评税年度享受扣税，需确保供款在 3 月 31 日前到账。

**Q4：如果我中途停止供款 QDAP 或 TVC，已扣的税需要退还吗？**

**A4**：不需要。税务扣除是基于您在特定评税年度内实际支付的保费或供款。后续是否继续供款不影响已获得的税务宽减。但请注意，提前中止 QDAP 可能会有财务损失，而 TVC 账户中的资金则会继续根据您选择的基金进行投资，直至退休。

### 3. 居所贷款利息扣除（Home Loan Interest Deduction）：将按揭成本转化为税务宽减

为购买自住物业而支付的按揭贷款利息，是香港一项重要的税务扣除项目。它允许纳税人在指定年期内，从其应课税入息中扣除部分利息支出。

**核心优势**：直接降低高额按揭利息带来的财务压力，是置业人士最直接有效的节税工具之一。

#### **关键要素速览**

|   |   |
|---|---|
|**要素**|**规则与说明**|
|**扣除上限**|• **每个课税年度最高 HK$100,000**（截至 2024/25 评税年度）。 • 自 2024/25 课税年度起，若纳税人与新生子女同住，该年度的扣除上限提高至 **HK$120,000**。|
|**扣除年期**|• **累计最多 20 个课税年度**。这 20 年可以是连续的，也可以是不连续的。|
|**适用物业**|• 必须是位于香港的住宅物业。 • 在该课税年度内，该物业必须全部或部分用作纳税人或其配偶的**主要居所**。|
|**合规要点**|1. **仅限利息**：只有按揭贷款的**利息部分**可以扣除，偿还的本金、手续费、行政费等均不可扣除。<br>    <br>2. **自住要求**：如果物业在一年中部分时间出租、部分时间自住，则只有自住期间的利息可以按比例扣除。<br>    <br>3. **物业拥有权**：纳税人必须是该物业的拥有者（独有或联权/分权共有）。<br>    <br>4. **不可与租金扣除重叠**：在同一个课税年度内，纳税人不能同时就同一物业申索居所贷款利息扣除和住宅租金扣除。<br>    <br>5. **证明文件**：须保留由贷款机构（如银行）发出的年度还款结单，上面会清楚列明年度内支付的利息总额。|

#### **夫妻共有物业的分配逻辑**

当物业由夫妻二人共同拥有时，利息扣除的分配方式取决于他们的评税方式和业权形式。

|   |   |   |
|---|---|---|
|**业权与评税方式**|**利息扣除分配规则**|**示例**|
|**联权共有（Joint Tenant）**|• **分开评税**：每人可申索的最高扣除额为 HK$$100,000 的一半，即 **H$$50,000**。<br><br>• **合并评税**：二人可共同申索最高 **HK$100,000** 的扣除额。|夫妻联权拥有一物业，年利息支出 HK$150,000。<br><br>• 分开评税：每人最多扣 HK$$50,000，合计 H$$100,000。<br><br>• 合并评税：合计扣除 HK$100,000。|
|**分权共有（Tenant in Common）**|按业权比例分配 HK$100,000 的最高扣除额。|丈夫占 70% 业权，妻子占 30%。<br><br>• 丈夫最高可扣 HK$70,000。<br><br>• 妻子最高可扣 HK$30,000。|
|**一方为业主，另一方非业主**|只有作为业主的配偶才有资格申索利息扣除。但如果选择**合并评税**，业主配偶支付的利息可以在合并后的总入息中扣除，从而惠及另一方。|妻子是唯一业主，年利息支出 HK$80,000。<br><br>• 分开评税：只有妻子能申索 HK$80,000 扣除。<br><br>• 合并评税：可在总收入中扣除 HK$80,000。|

**重点提示**：对于共同拥有物业的夫妻，如果一方收入远高于另一方，或者一方没有收入，选择**合并评税**通常是最大化利息扣除效益的最佳策略。

#### **常见问答（FAQ）**

**Q1：20 年的扣除期用完了怎么办？**

**A1**：一旦纳税人就某个物业累计使用了 20 个课税年度的利息扣除，便不能再就该物业或任何其他物业申索此项扣除。因此，在收入较低、税阶不高的年份，可以考虑暂时不申索，将宝贵的扣除年期留给未来收入和税率更高的年份使用。

**Q2：如果我换楼了，新的按揭利息可以重新计算扣除年期吗？**

**A2**：不可以。20 年的扣除限额是跟**纳税人**挂钩的，而非物业。无论您更换多少次物业，一生中可申索居所贷款利息扣除的总年期就是 20 年。

**Q3：我的物业部分用于出租，部分自住，利息如何计算扣除？**

**A3**：您需要按比例计算自住部分的利息。例如，如果物业一半面积自住、一半出租，则只有 50% 的按揭利息可用于申索居所贷款利息扣除。另外 50% 的利息可作为计算物业出租的应评税净值时的一项扣除。

**Q4：“与新生子女同住”的额外扣除上限如何界定？**

**A4**：从 2024/25 课税年度起，如果纳税人在该年度内与 2023 年 10 月 25 日或之后出生的子女连续同住至少 6 个月，其居所贷款利息的扣除上限将由 HK$$100,000 提高至 H$$120,000。此项额外宽减的年期上限为 19 年。

---

### 4. 住宅租金扣除（Domestic Rent Deduction）：租房一族的节税新选择

自 2022/23 评税年度起，香港为租住私人住宅的纳税人引入了一项新的税务宽减——住宅租金扣除，旨在减轻无持有任何住宅物业人士的租金负担。

**核心优势**：为庞大的“租楼”群体提供了直接的税务减免，每年最高可扣除十万港元的租金开支，显著降低应课税入息。

#### **关键要素速览**

|   |   |
|---|---|
|**要素**|**规则与说明**|
|**扣除上限**|• **每个课税年度最高 HK$100,000**（截至 2024/25 评税年度）。 • 自 2024/25 课税年度起，若纳税人与新生子女同住，该年度的扣除上限提高至 **HK$120,000**。|
|**适用人群**|• 必须是薪俸税或个人入息课税的纳税人。 • 纳税人或其同住配偶在香港**未持有任何住宅物业**。|
|**合资格租约**|• 必须是**书面订立的租约**。 • 租约必须已根据《印花税条例》**加盖印花（“打厘印”）**。|
|**合规要点与限制**|1. **主要居所**：所租住的物业必须是纳税人在香港的**主要居所**。<br>    <br>2. **不可重复申索**：不能与“居所贷款利息扣除”在同一期间重复享受。如果纳税人在一年内先租后买（或先卖后租），可以分段申索，但总扣除额不得超过年度上限。<br>    <br>3. **与雇主津贴冲突**：如果雇主提供了居所或报销了部分/全部租金，则不可申索此项扣除。<br>    <br>4. **关联方交易限制**：如果业主是纳税人或其配偶的关联人士（如父母、子女、兄弟姐妹、合伙人或由其控制的公司），则该租约下的租金不可扣除。<br>    <br>5. **分租限制**：如果将部分物业分租出去，则不可申索扣除。<br>    <br>6. **公屋限制**：公屋单位的租户或认可住客不可申索此项扣除。|

#### **夫妻及合租的分配逻辑**

租金扣除的分配规则，旨在确保公平并防止滥用。

|   |   |   |
|---|---|---|
|**租住情况**|**租金扣除分配规则**|**示例**|
|**夫妻共同租住**|• 如果只有一人是租客，则由该租客申索全额扣除。<br><br>• 如果夫妻二人均为租客，则他们需协议由其中一人申索全额扣除，或按商定比例分配，总额不超过 HK$100,000。如无协议，则平分扣除额。|夫妻共同租住，年租金 HK$150,000。<br><br>• 他们可以协议由丈夫申索全部 HK$$100,000，或每人申索 H$$50,000，或任何其他组合，总和不超过 HK$100,000。|
|**与他人合租**|最高扣除额 HK$100,000 会根据合租租客的人数按比例减少。|你与另外两位朋友（共 3 人）合租一套公寓，租约上列明你们三人为共同租客。<br><br>• 你的个人最高扣除上限为 HK$$100,000 / 3 = **H$$33,333**。|

**重点提示**：确保您的租约已**正式加盖印花**是申索此项扣除的绝对前提。未经打厘印的租约不符合资格。此外，务必厘清与业主的非关联关系，以避免合规风险。

#### **常见问答（FAQ）**

**Q1：如果我只租了半个课税年度，扣除上限是多少？**

**A1**：扣除上限会按租期比例计算。例如，如果您只租了 6 个月（即半个课税年度），您的最高扣除上限即为 HK$$100,000 / 2 = **H$$50,000**。您可扣除的金额为您这 6 个月实际支付的租金与 HK$50,000 之间的较低者。

**Q2：我配偶在内地有房产，我在香港租房，可以申请租金扣除吗？**

**A2**：可以。限制条件是纳税人或其同住配偶在**香港境内**持有住宅物业。如果物业位于香港以外地区，则不影响您在香港申索住宅租金扣除的资格。

**Q3：租约中包含管理费和差饷，这部分可以扣除吗？**

**A3**：不可以。住宅租金扣除仅限于为租用物业而支付的“租金”本身。租约中明确列出的管理费、差饷、水电煤等杂费开支，均不符合扣除资格。

**Q4：我和男朋友/女朋友同居租房，但租约只写了我一个人的名字，他/她可以扣税吗？**

**A4**：不可以。只有租约上列明的租客才有资格申索扣除。在这种情况下，只有您可以申索。如果你们是已婚夫妻，即使只有一方是租客，另一方也可以根据夫妻分配规则享受扣除。

## 二、免税额最大化策略

除了主动申请的“扣除”项目，充分利用法律赋予的各项“免税额”是降低应课税入息的基础。本章节将重点解析如何通过合理的家庭财务安排，最大化利用各类免税额，并探讨夫妻合并计税的省税逻辑。

### 1. 家庭相关免税额：善用家庭结构优势

香港税制为照顾家庭成员提供了多种免税额，其中最主要的是供养父母/祖父母免税额、子女免税额及单亲免税额。

#### **供养父母/祖父母/外祖父母免税额**

纳税人供养父母、祖父母或外祖父母，在满足特定条件下可申索此项免税额。关键区别在于受养人年龄及是否与纳税人同住。

**关键要素速览（截至 2024/25 评税年度）**

|   |   |   |
|---|---|---|
|**免税额类别**|**每名受养人免税金额**|**核心适用条件**|
|**基本免税额（55-59 岁）**|HK$25,000|1. 受养人年龄介于 55 至 59 岁。<br>    <br>2. 通常在香港居住。<br>    <br>3. 纳税人或其配偶在该年度内至少连续 6 个月与其同住，无须为他们付出金钱；**或**<br>    <br>      纳税人或其配偶每年给予不少于 **HK$12,000** 的金钱作生活费。|
|**基本免税额（60 岁或以上）**|HK$50,000|1. 受养人年满 60 岁，或未满 60 岁但有资格根据政府伤残津贴计划申领津贴。<br>    <br>2. 其他条件同上。|
|**额外免税额（与纳税人同住）**|在上述基本免税额基础上，**额外增加一倍**。<br><br>• 55-59 岁：HK$25,000<br><br>• 60 岁或以上：HK$50,000|1. 必须满足申领基本免税额的所有条件。<br>    <br>2. 在该课税年度内，受养人与纳税人**全年**持续同住，无须付出十足费用。|
|**伤残受养人免税额**|HK$75,000|如果受养人有资格根据政府伤残津贴计划申领津贴，纳税人可申索此项免税额，以代替供养父母/祖父母免税额。|

**合并计算示例**：

- **供养一位 65 岁且全年同住的父亲**：可申索金额 = HK$$50,000 (基本) + H$$50,000 (额外) = **HK$100,000**。
    
- **供养一位 58 岁但非同住的母亲（每年给予超过 HK**$$12,000 生活费）**：可申索金额 = **H$$**25,000**。
    

**合规要点与限制：**

- **不可重复申索**：就同一位受养人，所有纳税人（例如兄弟姐妹之间）只能由**一人**申索免税额。如有争议，税务局有权不批准任何一方的申索。
    
- **与安老院舍开支扣除的互斥关系**：如果纳税人已就某位父母/祖父母申索了“长者住宿照顾开支”扣除，则**不能**再就该位受养人申索供养父母/祖父母免税额（包括额外免税额）。纳税人需比较哪种方式更为节税。通常，如果符合条件，供养免税额的金额更高。
    
- **举证责任**：税务局可能要求纳税人提供证明文件，如受养人的身份证明、住址证明（以证实同住），或银行转账记录、书面确认书等（以证实给予生活费）。
    

#### **子女免税额**

为每名受供养的未婚子女，纳税人可获得子女免税额。

**关键要素速览（截至 2024/25 评税年度）**

|   |   |   |
|---|---|---|
|**免税额类别**|**每名子/女免税金额**|**核心适用条件**|
|**基本子女免税额**|HK$130,000|1. 子女未婚。<br>    <br>2. 年龄未满 18 岁；或年满 18 但未满 25 岁，并在大学、学校或其他类似教育机构接受**全日制教育**；或年满 18 岁，但因身体或精神问题无能力工作。|
|**出生年度额外免税额**|**额外增加** HK$130,000|在子女出生的该个课税年度，可额外获得此项免税额。|

**合规要点与限制：**

- **夫妻间分配**：子女免税额必须由夫妻其中**一方**申索全部。通常建议由收入较高、税阶较高的一方申索，以获得更大的税务利益。
    
- **非学费扣除**：子女免税额是对供养子女行为的宽减，与实际支付的学费、托儿所费用等无关。香港薪俸税下并无独立的学费扣除项目。
    
- **离婚或分居父母**：税务局会根据双方对子女的贡献（抚养费、教育开支等）来决定由谁申索或如何分配免税额。
    

#### **单亲免税额**

此项免税额旨在为独力或主力抚养子女的单身、丧偶或分居人士提供额外税务宽减。

**关键要素速览（截至 2024/25 评税年度）**

|   |   |   |
|---|---|---|
|**免税额类别**|**免税金额**|**核心适用条件**|
|**单亲免税额**|HK$132,000|1. 纳税人在该课税年度内**不属于已婚人士**（即单身、丧偶、离婚或全年与配偶分开居住）。<br>    <br>2. 纳税人在该年度内**独力或主力抚养**任何根据规定有权申索子女免税额的子女。<br>    <br>3. 必须先符合申索子女免税额的资格。|

**合规要点与限制：**

- **不适用于已婚人士**：即使配偶无收入，只要是已婚状态，就不能申索单亲免税额，而应申索“已婚人士免税额”。
    
- **离婚/分居当年不可申索**：在办理离婚或分居手续的该个课税年度内，不可申索单亲免税额。从下一个完整的课税年度开始方可考虑。

### 2. 合并计税（Joint Assessment）：夫妻同心，税负更轻

对于已婚夫妇，香港税法提供了“合并评税”这一选项。虽然默认情况下夫妻双方是分开评税的，但在特定条件下，选择合并评税可以有效降低家庭的总税负。

**核心逻辑**：合并评税将夫妻二人的入息总额合并计算，然后扣除二人合计可享有的免税额和扣除项目。这使得一方未用尽的免税额或扣除额，可以“转移”给另一方使用，从而降低高收入一方的应课税入息。

#### **何时合并计税更划算？**

提供一个简易的判定逻辑：当**夫妻一方的入息净额（即入息减去扣除和免税额）低于其个人应享有的免税额总和**时，选择合并评税几乎总是有利的。这通常发生在以下几种情况：

- **一方收入较低或无收入**：这是最典型的情景。低收入方的免税额（如基本免税额）未被完全使用。
    
- **一方有大量扣除项目**：例如，一方支付了高额的个人进修开支、居所贷款利息或认可慈善捐款，导致其应课税入息显著降低，甚至低于其免税额。
    
- **转移未用尽的免税额**：合并评税的核心就是将被“浪费”的免税额额度，用于抵消另一方的高额收入。
    

#### **直观对比：分开计税 vs. 合并计税**

假设陈先生年收入 HK$$600,000，陈太太年收入 H$$120,000。他们没有子女或其他供养人士。我们仅考虑基本免税额（每人 HK$132,000，截至 2024/25 评税年度）。

**情景一：分开评税**

|   |   |   |
|---|---|---|
|**项目**|**陈先生**|**陈太太**|
|应课税入息|HK$600,000|HK$120,000|
|减：基本免税额|(HK$132,000)|(HK$132,000)|
|**应课税入息实额**|**HK$468,000**|**HK$0** (入息低于免税额)|
|应缴税款（估算）|HK$63,560|HK$0|
|**家庭总税负：HK$63,560**|   |   |

**注意：陈太太有 HK$12,000 (132,000 - 120,000) 的免税额未被使用。**

**情景二：合并计税**

|   |   |
|---|---|
|**项目**|**合并计算**|
|总入息 (600,000 + 120,000)|HK$720,000|
|减：已婚人士免税额|(HK$264,000)|
|**应课税入息实额**|**HK$456,000**|
|应缴税款（估算）|HK$61,520|
|**家庭总税负：HK$61,520**|   |

**结论**：在此案例中，通过合并计税，家庭总税负减少了 **HK$2,040** (63,560 - 61,520)。

#### **申报要件与路径**

- **资格**：纳税人必须已婚，且并非与配偶分居。夫妻二人中，必须至少有一人有应评税的入息。
    
- **如何申报**：在填写个人报税表（BIR60）时，在相关部分选择“合并评税”。夫妻双方只需在**各自的报税表上**都作出此选择即可。税务局在收到双方的报税表后，会判断合并评税是否对纳税人有利。如果有利，便会自动以合并方式计算税款；如果不利，则仍会按分开评税处理，确保纳税人不会因此多缴税。
    

**智能评估**：纳税人无需自行精确计算哪种方式更优。只要符合条件并提出申请，税务局的系统会自动采用对纳税人最有利的评税方式。因此，在不确定的情况下，大胆选择合并评税是一个“有益无害”的操作。

## 三、其他合规节税手段

除了主流的扣税工具和免税额，香港税法还提供了一些其他途径，帮助个人和企业主在合规的前提下，进一步优化税务负担。

### 1. 慈善捐款（Approved Charitable Donations）

向税务局认可的慈善机构作出的捐款，可以申请扣除。

|   |   |
|---|---|
|**要素**|**规则与说明**|
|**扣除上限**|• **年度可课税入息/利润的 35%**。|
|**最低门槛**|• 捐款合计总额不少于 **HK$100**。|
|**合规要点**|1. **认可机构**：捐款必须是给予名列《税务条例》第88条下的获豁免缴税的慈善机构或慈善信托。您可以在税务局网站查阅这份名单。<br>    <br>2. **收据要求**：必须保留由认可慈善机构发出的正式收据作为凭证。<br>    <br>3. **仅限金钱**：捐赠物资（如衣物、食品、艺术品）的价值**不可**用作扣税。<br>    <br>4. **不可为利益交换**：购买慈善奖券、电影票、参加慈善晚宴或拍卖物品的款项，通常不被视为纯粹捐款，不可扣除。|

**节税效果**：此项扣除的节税效果直接与您的边际税率挂钩。若您的最高边际税率为 17%，则每捐赠 HK$$1,000，可节省 H$$170 的税款。

### 2. 个人进修开支（Self-education Expenses）

为提升工作技能或获取职业资格而支付的进修费用，可以申请扣除。

|   |   |
|---|---|
|**要素**|**规则与说明**|
|**扣除上限**|• **每个课税年度最高 HK$100,000**（截至 2024/25 评税年度）。|
|**合资格课程**|• 必须是为获取或维持在现有或未来工作中的**职业资格**而修读的课程。 • 课程须由合资格的教育机构提供，包括行业协会、大学、专业团体等。|
|**可扣除费用**|• 课程费用、考试费用。 • 某些情况下，与课程直接相关的书本费也可能被接纳。|
|**合规要点**|1. **与工作相关**：课程必须与纳税人当前或计划从事的职业有直接关系。纯为个人兴趣而修读的课程（如烹饪、插花）不可扣除。<br>    <br>2. **雇主报销处理**：任何已获雇主、政府（如持续进修基金）或其他方面报销的费用，均须在申索额中剔除。<br>    <br>3. **证明文件**：保留学费收据、课程大纲、考试成绩单等，以证明开支的真实性及与工作的相关性。|

### 3. 企业主特有：利得税下的可扣除开支

对于小微企业主和自雇人士而言，利得税的规划核心在于准确界定和申报所有“为赚取应评税利润而付出”的经营开支。

**核心原则**：根据《税务条例》，所有在产生应评税利润过程中**完全（Wholly）、纯粹（Exclusively）及必需（Necessarily）**的开支，均可获准扣除。

#### **常见可扣除的经营开支**

- **居家办公（Home Office）开支**：如果部分居所被用作办公室，可按合理基准（如按面积比例）分摊部分租金、水电、物业管理费等作为经营开支。需备存计算依据。
    
- **商务差旅与应酬**：与业务相关的交通、住宿、餐饮费用。须保留详细单据（发票、收据）并能证明其商业目的（如会议记录、客户拜访行程）。
    
- **员工成本**：支付给雇员的薪金、津贴、强积金供款等。
    
- **固定资产折旧（折旧免税额）**：购买用于业务的资产（如电脑、办公家具、机器）不能一次性全额扣除，但可以按税务局规定的折旧率，分年获得“折旧免税额”。
    
- **可即时扣除的特定项目**：
    
    - **电脑软硬件**：购买用于业务的电脑硬件和软件，可获 **100% 即时扣除**。
        
    - **环保设备**：购买合资格的环保设施和机械，可获 **100% 即时扣除**。
        
    - **研发（R&D）开支**：合资格的研发开支可享有**超级扣减（Super Deduction）**，首 HK$200 万的合资格开支可获 300% 扣除，余额可获 200% 扣除。
        

#### **两级利得税制与涉税风险**

- **两级制优惠**：自 2018/19 年度起，法团（公司）及非法团业务（独资/合伙）可享受两级利得税率。
    
    - **首 HK$200 万**应评税利润：法团税率 **8.25%**，非法团业务税率 **7.5%**。
        
    - **超出部分**：恢复至标准税率，法团 **16.5%**，非法团业务 **15%**。
        
    - **注意**：关联企业中只有一个能提名享受此优惠。
        
- **常见涉税风险**：
    
    - **公私不分**：将个人开支混入公司账目作扣除，是审计的重点审查对象。
        
    - **单据不全**：无法为大额开支提供合规单据，可能导致该项扣除被税务局挑战。
        
    - **资本性开支误作收益性开支**：例如，将物业装修的大部分费用一次性扣除，而不是作为资本改良处理。

## 四、节税组合方案建议（含示例）

理解了各项工具后，关键在于如何将它们组合起来，以适应不同的个人和家庭情况。本章节将通过两个典型场景，演示如何配置节税方案，并估算其潜在的税务节省。

**重要声明**：以下所有计算均为**估算示例**，旨在说明节税原理。计算均基于 2024/25 评税年度的税率及免税额。实际节税额取决于您的个人具体情况、税务局的最终评税，以及未来政策的变动。估算采用了**边际税率法**，即假设新增的每一元扣除额，都是在您最高的税阶上产生节税效应。

### 1. 案例一：高薪单身人士

**背景假设：**

- **李先生**：单身，30 岁，无子女。
    
- **年收入**：HK$$960,000 (月薪 H$$80,000)。
    
- **家庭情况**：父母均已满 60 岁，非同住，李先生每年给予每位父母超过 HK$12,000 的生活费。
    
- **居住情况**：在市区租房，月租 HK$15,000。
    
- **个人规划**：希望为退休储蓄，并提升专业技能。
    

**节税方案配置与估算：**

|   |   |   |   |
|---|---|---|---|
|**节税项目**|**配置策略与说明**|**可扣除/免税金额**|**边际税率节税估算 (17%)**|
|**基本免税额**|个人基本免税额。|HK$132,000|_（基础项，不计入额外节税）_|
|**供养父母免税额**|为父亲和母亲分别申索供养年满60岁父母免税额。|HK$$50,000 × 2 = **H$$****100,000</b>**|HK$$100,000 × 17% = **H$$****17,000</b>**|
|**住宅租金扣除**|年租金支出 HK$180,000，申索扣除上限。|**HK$100,000**|HK$$100,000 × 17% = **H$$****17,000</b>**|
|**QDAP/TVC 组合**|每年投入 HK$60,000 到 TVC 账户，利用其投资灵活性。|**HK$60,000**|HK$$60,000 × 17% = **H$$****10,200</b>**|
|**自愿医保 (VHIS)**|为自己及父母三人购买 VHIS，假设每人保费均超过 HK$8,000。|HK$$8,000 × 3 = **H$$****24,000</b>**|HK$$24,000 × 17% = **H$$****4,080</b>**|
|**个人进修开支**|报读一个与工作相关的硕士课程，年学费 HK$80,000。|**HK$80,000**|HK$$80,000 × 17% = **H$$****13,600</b>**|
|**慈善捐款**|每年向认可慈善机构捐款。|HK$10,000 (假设)|HK$$10,000 × 17% = **H$$****1,700</b>**|
|**合计年度节税估算**|   |   |**HK$63,580**|

**方案小结**：通过全面配置各项扣除和免税额，李先生的应课税入息总共减少了 **HK**$$374,000** (100,000 + 100,000 + 60,000 + 24,000 + 80,000 + 10,000)，从而每年大约节省了超过 **H$$**63,000** 的税款。

### 2. 案例二：双薪家庭（合并计税）

**背景假设：**

- **王先生**：年收入 HK$700,000。
    
- **王太太**：年收入 HK$300,000。
    
- **家庭结构**：育有一名 5 岁子女。王先生的父亲（68岁）与他们全年同住。
    
- **居住情况**：共同拥有一套自住物业（联权共有），每年按揭利息支出 HK$140,000。
    
- **个人规划**：希望为家庭提供全面保障，并最大化家庭的税务节省。
    

**计税方式决策：分开计税 vs. 合并计税**

由于夫妻双方收入差距不是特别悬殊，且都有稳定收入，需要简单测算才能确定。但考虑到他们有共同的居所贷款利息需要扣除，**合并计税**通常能更好地利用扣除额。我们将以合并计税为例进行方案配置。

**节税方案配置与估算（合并计税）：**

|   |   |   |   |
|---|---|---|---|
|**节税项目**|**配置策略与说明**|**可扣除/免税金额**|**边际税率节税估算 (17%)**|
|**已婚人士免税额**|合并计税下自动适用。|HK$264,000|_（基础项）_|
|**子女免税额**|为 5 岁子女申索。|**HK$130,000**|HK$$130,000 × 17% = **H$$****22,100</b>**|
|**供养父母免税额**|为同住的父亲申索“基本+额外”免税额。|HK$$50,000 + H$$50,000 = **HK$100,000**|HK$$100,000 × 17% = **H$$****17,000</b>**|
|**居所贷款利息**|物业为联权共有，合并计税下可扣除全部利息，上限为 HK$100,000。|**HK$100,000**|HK$$100,000 × 17% = **H$$****17,000</b>**|
|**QDAP/TVC 组合**|夫妻二人共同规划，最大化利用 HK$120,000 额度。 • 王先生购买 QDAP，年缴 HK$60,000。 • 王太太购买 QDAP，年缴 HK$60,000。 合并计税下可全额扣除。|HK$$60,000 + H$$60,000 = **HK$120,000**|HK$$120,000 × 17% = **H$$****20,400</b>**|
|**自愿医保 (VHIS)**|为一家四口（夫妻、子女、父亲）购买，假设每人保费均超 HK$8,000。|HK$$8,000 × 4 = **H$$****32,000</b>**|HK$$32,000 × 17% = **H$$****5,440</b>**|
|**合计年度节税估算**|   |   |**HK$81,940**|

**方案小结与易踩坑点：**

- **优化分配**：通过合并计税，家庭不仅充分利用了居所贷款利息和 QDAP 的总扣除上限，还将所有免税额集中使用，有效降低了家庭的整体应课税入息。
    
- **QDAP 优于 TVC**：在此案例中，如果王太太自己购买 TVC，她的低收入可能不足以完全利用 HK$60,000 的扣除额。而购买 QDAP，则可以在合并计税时将她支付的保费扣除额“转移”到家庭总收入中，实现完美利用。
    
- **关键合规点**：
    
    - **分开评税的陷阱**：如果选择分开评税，由于是联权共有物业，每人最多只能扣 HK$$50,000 的利息，总计还是 H$$100,000，但分配受限。更重要的是，王太太可能因自身收入不高而无法用尽她的扣除和免税额度。
        
    - **子女免税额归属**：分开评税时，必须协议由一方（通常是收入高的一方，即王先生）申索子女免税额，不可拆分。
        
    - **供养父母免税额的唯一性**：如果王先生有兄弟姐妹，需确保只有王先生一人申索其父亲的免税额。

## 申报准备清单

为了顺利完成年度报税并确保所有合资格的扣除和免税额都被申领，建议您提前准备好以下文件和信息。虽然提交报税表时无需附上所有证明文件，但您必须将它们妥善保存至少 6 年，以备税务局日后查验。

**个人与家庭信息**

- 个人及配偶的香港身份证号码及个人资料。
    
- 子女的出生证明书。
    
- 受供养父母/祖父母/兄弟姐妹的身份证明文件及年龄证明。
    

**入息证明**

- 由雇主填报的《雇主报税表》（IR56B 表格）副本。
    
- 全年所有粮单、花红及津贴通知书。
    
- 如有其他收入（如租金、独资业务利润），备妥相关账目和记录。
    

**扣除项目证明文件**

- **自愿医保计划 (VHIS)**
    
    - 保险公司发出的“保费收据”及“年度保单结单”。
        
- **合资格延期年金 (QDAP) / 强积金自愿性供款 (TVC)**
    
    - 保险公司发出的“合资格延期年金保费的年度摘要”。
        
    - 强积金受托人发出的“可扣税自愿性供款概要”。
        
- **居所贷款利息**
    
    - 贷款机构（如银行）发出的“年度还款结单”，清楚列明年度内支付的利息总额。
        
- **住宅租金扣除**
    
    - 已加盖印花（打厘印）的住宅租约。
        
    - 全年的租金收据或银行转账记录。
        
- **认可慈善捐款**
    
    - 由《税务条例》第88条认可的慈善机构发出的正式捐款收据。
        
- **个人进修开支**
    
    - 课程及考试的缴费收据。
        
    - 能证明课程与工作相关的资料，如课程大纲或由雇主出具的证明信。
        
- **长者住宿照顾开支**
    
    - 由安老院舍发出的年度缴费证明。
        

**重要时间节点**

- **每年 5 月初**：通常会收到税务局发出的个人报税表（BIR60）。
    
- **报税表发出后 1 个月内**：个人报税的截止日期。
    
- **报税表发出后 3 个月内**：独资业务的东主的报税截止日期。
    
- **网上报税**：通过“税务易”（eTAX）账户报税，通常可自动获延长 1 个月的报税期限。
    

**最后的提醒**：养成在日常生活中系统性整理和归档上述单据的习惯，将使您在每年的报税季从容不迫，确保不会因遗漏任何文件而错失应得的税务宽减。
