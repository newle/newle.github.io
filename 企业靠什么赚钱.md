# 盈利
1年内的盈利 = (销售价 - 成本) * 翻台次数
成本包括：原料成本 + 销售成本
- 提高销售价，取得定价权，定位明确。
- 降低成本，优化渠道，优化方案，优化
- 提高交易完成速度，交易过程的润滑程度，进而提高翻台次数
