# 风险
- 保险主要针对纯粹风险（无收益的），和特定风险（非普遍的）。
- 保险主要针对财务风险和身体风险。情绪风险不可保。
- 保险主要是为了将风险转移，其他辅助好处是提供就业机会、金融服务、损失控制、储蓄和投资、经济增长。
# 法律原则
- 合约法 / 代理法
- 保险合约是简单合约，无需盖章。  契据是书面的，比如转让土地，担保等，需要盖章签署。
